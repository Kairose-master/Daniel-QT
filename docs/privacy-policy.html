다니엘과 세친구

개인정보처리방침

최종 업데이트: 2026-07-23

"다니엘과 세친구"(이하 "서비스")는 소그룹 큐티(QT) 나눔을 위한 애플리케이션입니다. 본 방침은 서비스가 어떤 정보를 수집하고 어떻게 이용·보관·파기하는지 설명합니다.

1. 수집하는 정보

  • 계정 정보: 이메일 주소, 이름(별명), 프로필 이미지(선택). 카카오 로그인 사용 시 카카오가 제공하는 닉네임·프로필 이미지.
  • 활동 정보: 묵상 글(느낀점·기도제목), 댓글, 위로 스탬프, 출석 기록.
  • 응원 음성: 이용자가 직접 녹음하여 같은 소그룹 지체에게 전송한 오디오 파일.
  • 알림 토큰: 푸시 알림 발송을 위한 기기 토큰.

2. 이용 목적

수집한 정보는 소그룹 나눔 기능 제공, 참여 현황(출석·연속 묵상) 집계, 푸시 알림 발송에만 사용합니다. 광고 목적으로 사용하거나 제3자에게 판매하지 않습니다.

3. 보관 및 파기

정보는 서비스 이용 기간 동안 보관합니다. 이용자가 앱 내 내 설정 > 계정 삭제를 실행하면 관련 데이터(묵상·댓글·응원 음성 등)가 함께 삭제됩니다. 소그룹에서 나가면 해당 소그룹의 데이터에 대한 접근 권한이 사라집니다.

4. 위탁 및 제3자 처리

서비스는 다음 사업자에게 데이터 처리를 위탁합니다. 이들은 목적 외 사용이 금지됩니다.

  • Supabase Inc. — 데이터베이스·인증·파일 저장(클라우드 인프라)
  • Expo (Expo Push) — 푸시 알림 발송
  • Anthropic — 관리자(리더)가 AI 본문 작성 기능을 사용할 때, 해당 소그룹 리더의 API 키로 묵상 안내 글을 생성

5. 이용자의 권리

이용자는 언제든 자신의 정보를 조회·수정할 수 있으며, 내 설정 > 계정 삭제를 통해 모든 정보의 삭제를 요청할 수 있습니다.

6. 아동의 개인정보

본 서비스는 만 14세 미만 아동을 주 대상으로 하지 않습니다. 만 14세 미만 아동이 이용할 경우 법정대리인의 동의가 필요합니다.
